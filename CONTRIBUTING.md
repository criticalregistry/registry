# Contributing

Corrections and additions are welcome.

**To correct a factual error:** open an issue with the specific data point, your correction, and a source.

**To add a project:** open a pull request adding a new file to `registry/` using the existing entries as a template. Include at least one verifiable source for each data point.

**Named individual data:** all entries containing named individuals are verified before publication. If you are named in an entry and wish to correct a factual error, contact contact@criticalsoftwareriskregister.com directly.
